# TrackerApp

מערכת לימוד וחזרות מקומית ללימוד תורה, מבוססת `C# WinForms + SQLite`.

## המודל העדכני

המוצר כבר אינו בנוי סביב כרטיסי `שאלה / תשובה` כלליים, אלא סביב `יחידת לימוד`.

לכל יחידה יש:

- נושא
- קישור לצומת בספרייה
- מקור
- פשט
- קושיה
- תירוץ
- חידוש
- סיכום אישי
- הערות חזרה
- תגיות
- נתוני תזמון וחזרה

## תאימות למסד קיים

מסדי SQLite קיימים עוברים מיגרציה בטוחה:

- `Question -> SourceText`
- `Answer -> PersonalSummary`

העמודות הישנות נשמרות לצורכי תאימות, אבל ממשק המשתמש ותצוגת המוצר מכוונים כעת ליחידת לימוד.

## עקרונות חשובים

- עץ הספרייה נשמר
- lazy loading נשמר
- חיפוש, תגיות, חזרה, גיבוי/שחזור וייצוא נשמרים
- ממשק עברי RTL נשמר
- נתוני runtime, לוגים וקבצי עזר צריכים להישמר בתיקיית נתוני היישום, לא בתוך נתיב מכונה קשיח
